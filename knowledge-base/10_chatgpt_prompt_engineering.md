# ChatGPT Prompt Engineering

ChatGPT performs best with structured, explicit prompts.

## Strengths
- Instruction following
- Output formatting
- Prompt rewriting
- Code generation

## Best Practices
- Use clear step-by-step instructions
- Explicitly define output formats
- Use self-reflection prompts when accuracy matters

## Example Optimization Pattern
"Rewrite the prompt below to improve clarity, add missing constraints, and enforce a clear output structure."

## Weaknesses
- May hallucinate if context is weak
- Overconfident without verification prompts
