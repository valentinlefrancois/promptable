# Claude Prompt Engineering

Claude excels at reasoning, critique, and long-context understanding.

## Strengths
- Deep reasoning
- Long-form analysis
- Prompt critique
- Ethical and safety-aware outputs

## Best Practices
- Provide rich context
- Encourage step-by-step reasoning
- Explicitly allow uncertainty

## Example Pattern
"Analyze the prompt below, identify weaknesses, and propose an improved version."

## Weaknesses
- Less strict with output formats
- Can be verbose if unconstrained
